**Spacer**

Spacer는 Row 또는 Column과 같은 Flex 컨테이너 내 위젯 사이의 간격을 조정하는 데 사용할 수 있는 조정 가능한 빈 공간를 생성한다 .

```
 Row(
  children: [
    Text('Begin'),
    Spacer(flex: 4), // Flex의 기본값은 1이다.
    Text('Middle'),
    Spacer(flex: 2),
    Text('End'),
  ],
)

```


**Stack**

위젯의 자식 요소들을 부모위의 가장자리를 기준으로 배치하는 위젯이다.

```
Stack(
children: [
  Container(
   width: 100,
     height: 100,
   color: Colors.red,
  ),
  Container(
   width: 90,
   height: 90,
   color: Colors.green,
   ),
   Container(
    width: 80,
   height: 80,
     color: Colors.blue,
            ),
          ],
        ),
```

 텍스트와 이미지를 겹쳐 놓고 하단에 버튼을 추가하는 등 여러 자식 요소를 간단하게 겹쳐 표시하고 싶을 때 유용하다.
