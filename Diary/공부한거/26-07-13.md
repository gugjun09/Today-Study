# ott앱 만들기 강의 들음

프로필 추가 기능을 구현했다.



**Random**

수를 무작위로 뽑아주는 위젯이다. 

```
Random().nextInt(256) // 0부터 256까지 숫자 한개를 랜덤으로 뽑는다.
```


전역 변수란 특정 함수나 클래스에 속하지 않고 맨땅에 박은 변수이며 어디서든 모든 곳에서 수정하고 접근할 수 있는 특을 가지고 있다.

앱 전반에 공유해야 하는 사용자 로그인 정보 등을 다룰 때 사용된다.


```
String fruit = 'apple'
```

구현이 매우 간단하고 직관적이다.
