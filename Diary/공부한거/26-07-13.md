# ott앱 만들기 강의를 듣고 모르는 단어와 모델 개념들을 공부함

프로필 추가 기능을 구현했다.



**Random**

수를 무작위로 뽑아주는 위젯이다. 

```
Random().nextInt(256) // 0부터 256까지 숫자 한개를 랜덤으로 뽑는다.
```

## 전역 변수


전역 변수란 특정 함수나 클래스에 속하지 않고 맨땅에 박은 변수이며 어디서든 모든 곳에서 수정하고 접근할 수 있는 특징을 가지고 있다.

앱 전반에 공유해야 하는 사용자 로그인 정보 등을 다룰 때 사용된다.


```
String fruit = 'apple';
int number = '99';

```

구현이 매우 간단하고 직관적이다.

가장 단순한 반법으로 클래스 외부나 별도의 파일에 변수를 선언하여 어디서는 호출할 수 있다.







## model

model은 앱이 다루는 데이터의 구조와 상태를 정의하고 관리하는 핵심 요소이다. 

내가 배우고 있는 모델은 상태 모델이다(State/App Model)

상태 모델은 데이터를 추가, 삭제, 수정하는 로직을 포함하며, 데이터가 변경되면 화면에 이를 알린다.

```

class UserModel {
  final int id;
  String name;
  final int r;
  final int g;
  final int b;

  UserModel({
    required this.id,
    required this.name,
    required this.r,
    required this.g,
    required this.b,
  });

  factory UserModel.createUserModel() {
    return UserModel(
      id: DateTime.now().millisecondsSinceEpoch,
      name: '',
      r: Random().nextInt(256),
      g: Random().nextInt(256),
      b: Random().nextInt(256),
    );
  }

  factory UserModel.fromJson(Map<String, dynamic> json) {
    return UserModel(
      id: json['id'],
      name: json['name'],
      r: json['r'],
      g: json['g'],
      b: json['b'],
    );
  }

  Map<String, dynamic> toJson() {
    return {'id': id, 'name': name, 'r': r, 'g': g, 'b': b};
  }
}

```
