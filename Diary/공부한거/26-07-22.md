late 변수는 선언 시 값을 바로 할당하지 않고 나중에 초기화할 수 있도록 지연 처리하는 변수이다.

주로 초기화 값을 나중애 앱 실행 중에 받아야 할 때 사용된다.


JSON은 데이터 포맷일 뿐이며 단순히 데이터를 표시하는 표현방법이다. 그리고 JSON은 텍스트 데이터다.
텍스트 데이터는 String 형태로 들어온다.

역잘력화 : 서버에서 받은 JSON 문자열을 객채로 변환한다.
직렬화 : dart객채를 서버로 보낼 JSON문자열로 변환한다.

그리고 간단한 데이터 구조에서는 dart:convert 패키지 만으로 처리할 수가 있다.

실행하기 버튼 단축기는 Shift + f10 이다.


ListView.builder는 많은 양의 데이터를 터치해서 스크롤 가능한 목록으로 만들 때 사용하는 위젯이다.

itemCount : 리스트에 표시할 총 아이템의 갯수 (예시 : 데이터 배열의 길이)
itemBuilder : 화면에 리스트 항목을 그릴 때 호출되는 함수이다. index 형태를 가지며 각 index마다 어떤 위젯을 반환할지 정의한다.

 
indexedstack은 여러 자식 위젯 중 지정된 인덱스의 위젯 하나만 화면에 표시하는 위젯이다. 
화면을 전환할 때 기존 위젯들의 상태가 유지된다.
