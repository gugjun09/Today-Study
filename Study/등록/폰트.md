# 폰트 등록하는법

프로젝트 폴더 내에 asset/font/ 경로를 만들고 폰트 파일들을 넣은 뒤, pubspec.yaml 파일에 아래와 같이 등록한다.
```
  fonts:
     - family: Noto  // 원하는 이름
       fonts:
         - asset: asset/font/NotoSansKR-VariableFont_wght.ttf

     - family: Nanum  // 원하는 이름 
       fonts:
         - asset: asset/font/NanumMyeongjoBold.ttf
         weight: 700
```
패밀리 안에 여러개의 폰트 등록도 가능하다.\

**사용 예시**

```
Text(
   '나이를 입력해주세요.',
    style: TextStyle(
     color: Colors.white,
       fontFamily: 'Nanum',  // 패밀리 이름 넣으면 됨
       fontSize: 20,
      ),
 ),
```
