##  Class

클래서(class)는 멤버 변수와 멤버 함수를 가질 수 있으며, 클래스 명을 정할 때에는 첫 글자를 대문자로 시작하는 PascalCase 표기법을 사용해야 한다

```
class Person {
  String name = "tyger";
}
```
Person 클래스를 tyger라는 변수로 만들어 주고, 해당 객체가 사용 되어질 때에 instance 가 된다.

**(인스턴스)instance 란 메모리에 할당 되어지는 것을 의미한다.**

```
void main() {
  Person tyger = Person();
  print(tyger.name);
  // tyger
}

```

**(인스턴스)instance**

인스턴스란 Class를 기반으로 생성되며, Class의 기능과 속성을 동일하게 가지고 있는 프로그래밍 상에서 사용되는 대상이다.
